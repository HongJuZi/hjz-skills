# hjz-ui-rule-getter Skill

从上传的UI设计图片分析出完整的UI设计规范，生成标准化的设计方案文档。

## 功能特点

- **图片分析**：支持从本地图片或网络图片分析UI设计
- **全面分析**：分析色彩、字体、图标、布局等多个维度
- **标准化输出**：生成完整的UI设计规范文档
- **代码示例**：提供SCSS变量和组件样式示例
- **可重复使用**：适用于多个项目的UI设计规范提取

## 分析维度

1. **色彩系统**：主色调、辅助色、中性色、功能色
2. **字体规范**：字体家族、字号层级、字重规范
3. **图标规范**：尺寸、风格、颜色规则
4. **视觉效果**：渐变、阴影、圆角、透明度
5. **布局规范**：边距、间距、网格系统
6. **组件样式**：按钮、卡片、列表等组件

## 输出格式

- **Markdown格式**：完整的UI设计规范文档
- **SCSS变量**：色彩、字体、间距等变量定义
- **组件样式**：常用组件的SCSS样式示例

## 使用方法

1. **上传图片**：提供本地图片路径或网络图片URL
2. **分析图片**：系统自动分析图片中的设计元素
3. **生成规范**：生成标准化的UI设计规范文档
4. **应用规范**：将规范应用到项目开发中

## 应用场景

- **新项目启动**：从设计稿快速生成UI规范
- **现有项目重构**：统一视觉风格
- **设计评审**：检查设计一致性
- **团队协作**：统一设计语言

## 示例输出

```markdown
# UI设计规范

## 一：色彩规范

| 类别 | 项目 | 色值 | 用途 |
|------|------|------|------|
| 主色 | 主色 | `#1E88E5` | 顶部导航栏、按钮 |
| 辅助色 | 辅助色1 | `#4CAF50` | 成功状态、免费标签 |

## 二：字体规范

| 类别 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 大标题 | 20px | Bold | 页面主标题 |
| 中标题 | 16px | Medium | 模块标题 |

## 三：视觉效果

### 1. 渐变效果
- 起始色：`#1E88E5`
- 结束色：`#64B5F6`
- 方向：180deg

### 2. 阴影效果
- X偏移：0px
- Y偏移：2px
- 模糊半径：4px
- 颜色：`rgba(0, 0, 0, 0.05)`
```

## 技术实现

- **图片分析**：使用视觉识别技术分析图片中的设计元素
- **色彩提取**：从图片中提取主要色彩和色值
- **布局分析**：识别页面布局和组件结构
- **规范生成**：基于分析结果生成标准化的设计规范

此技能可以帮助设计和开发团队快速从设计稿中提取规范，确保设计的一致性和开发的准确性。
